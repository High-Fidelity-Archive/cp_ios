YOAuthSignatureMethod Protocol Reference

Conforms to NSObject
Declared in YOAuthSignatureMethod.h

Overview

A base protocol for all signature methods. This cannot be used as a signature method alone; instead use YOAuthSignatureMethod_PLAINTEXT or YOAuthSignatureMethod_HMAC-SHA1.
